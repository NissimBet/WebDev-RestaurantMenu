CSS

Existen 3 formas de agregar estilos a un archivo

  1. A traves de un archivo externo
  2. A traves del tag <style>
  3. A traves del atributo "style"

La forma en que se escriben las reglas de estilos es la misma para las 3 formas

1. Archivo externo

<link href="url" rel="stylesheet" type="text/css" />

2. Etiqueta style

<style> div { font-size: 16px } </style>

3. Atributo style

<div style="fontSize:16px;"> estilos </div>

Formas de seleccion de elementos

  • Por elemento

    div { }

  • Por id

    #id { }

  • Por clase

    .clase { }

Prioridades de seleccion de estilos

  • Por id
  • Por clase
  • Por elemento

Box-Model

Todos los elementos tienen un margen, un padding y un border

para el padding

padding: 20px; da 20 pixeles en todos direcciones padding: 20px 10px; da 20 pixeles hacia arriba y 10 hacia los lados padding: 20px 10px 5px 30px; da 20, 10, 5 y 30 pixeles hacia arriba, derecha, abajo e izquierda padding-top: 20px;
padding-right: 10px;
padding-bottom: 5px;
padding-left: 30px;
