CSS

Existen 3 formas de agregar estilos a un archivo

  1. A traves de un archivo externo
  2. A traves del tag <style>
  3. A traves del atributo "style"

La forma en que se escriben las reglas de estilos es la misma para las 3 formas

1. Archivo externo

<link href="url" rel="stylesheet" type="text/css" />

2. Etiqueta style

<style> div { font-size: 16px } </style>

3. Atributo style

<div style="fontSize:16px;"> estilos </div>

Formas de seleccion de elementos

  • Por elemento

    div { }

  • Por id

    #id { }

  • Por clase

    .clase { }

Prioridades de seleccion de estilos

  • Por id
  • Por clase
  • Por elemento

Algunos atributos

  • font-family

    Es el font del texto

  • font-size

    el tamano en pizeles del texto

  • color

    Puede ser un color predefinido (red, blue, green)

    Puede ser un numero en hexadecimal

  • border

    es el borde del elemento

    tiene un grosor, un estilo y un color

    border: 1px solid red

  • display

    Es la forma en que se muestra el elemento

    display: inline / inline-block / block / flex / grid / none

  • background

    es el fondo del elemento

    background: imageSrc repeat position-x position-y #color

  • width

    es el ancho del elemento

    width: #px / % / vw / vh

  • cursor

    es el tipo de cursor sobre el elemento

    cursor: pointer

  • opacity

    es la transparencia del elemento

    opacity: 0-1

  • text-transform

    convierte el texto en el elemento

    text-transform: lowercase / uppercase

  • float

    hace que un elemento flote sobre el resto

    float: left / right

Selectores

Las reglas de los elementos se pueden aplicar por medio de selectores

  1. elem1 elem2

    Selecciona todos los elem2 hijos de elem1

  2. elem1 > elem2

    Afecta a los hijos inmediatos

  3. elem[attrib="val"]

    Afecta a elementos con atributo val

  4. elem1, elem2

    Afecta a elementos ambos elementos

  5. elem1:hover

    Afecta al elemento cuando se le hace hover
