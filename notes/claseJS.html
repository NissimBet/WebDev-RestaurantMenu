Funciones

Existen 2 formas de declarar funciones

  • Con el Arrow function let nombre = () => {}
  • De la forma normal function nombre() {}

Se le pueden declarar valores default para los parametros de una funcion

function suma(p1 = 0; p2 = 0; p3 = 0) { return p1 + p2 + p3 }

algunos metodos son

  • arr.pop()

  • arr.splice()
  • arr.map()
  • arr.slice()
  • arr.reduce()
  • arr.find()
  • arr.indexOf()

Map

El map es un ciclo que no modifica el arreglo, aplicando una funcion sobre los elementos del arreglo original para regresar uno nuevo

let nums = [1,2,3,4];
let nuevo = nums.map(num => if(num % 2 === 0) { return num / 2; } ); Esto divide entre dos los pares del arreglo original

Filter

El filter genera un nuevo arreglo ejecutando una funcion sobre todos los elementos

let nums = [1,2,3,4];
let pares = nums.filter(num => num % 2 ); Esto regresa un arrego con los numeros pares

Splice

El splice tiene 3 parametros

  1. el index de donde se va a hacer algo
  2. La cantidad de elementos para remover
  3. Los elementos a agregar separados por coma
let nums = [1,2,3,4];
nums.splice(0, 1, 5, 6); Esto elimina el 1 y agrega un 5 y un 6 en ese lugar

Slice

El slice hace un substring entre 2 indices

let nums = [1,2,3,4];
nums.slice(0, 2); regresa los 2 primeros elementos

Objetos

Son diferentes a los objetos en otros lenguajes

let obj = {
nombre: 'nombre' apellido: 'apellido' edad: 10 }

Los elementos se pueden acceder de 2 maneras

  • obj.apellido
  • obj['apellido']

Cuando se declaran funciones como parte de un objeto, usar la notacion de:

function name() {}

ya que si se quiere hacer referencia a otro elemento del objeto, se puede usar this

que no se tiene en el scope de la arrow function

El DOM

Es el Document Object Model

Para cada elemento se genera un nodo, JavaScript lo genera automaticamente

Se representa como un arbol, donde los atributos son nodos hoja

document apunta a este arbol

Cada vez que se modifica dinamicamente el html, el DOM se modifica automaticamente

Eventos

Se deben asignar a un elemento especifico. No se puede asignar a una coleccion

elem.addEventListener('click', (event) => {})

existen muchos eventos, algunos:

  • click
  • change
  • submit
